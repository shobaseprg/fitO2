・制作背景
　プログラミングスクールを通う中で、多くの挫折者を目の当たりにした。挫折の理由は自信をなくしたことが理由であると感じる。最も効果的に自分の成長を感じることができ、尚且つ効率的に知識を定着させる方法は「他者に教える（アウトプットする）することであると考え、それを支援するためのアプリを作成しようと思い立った。
